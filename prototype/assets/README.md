## 设计规范

### 色彩系统

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色（校园蓝） | #409EFF | 主按钮、选中态、链接 |
| 主色深 | #337ECC | 按钮 hover、导航激活 |
| 成功色 | #67C23A | 已售/已解决/已成团 |
| 警告色 | #E6A23C | 待接单/进行中 |
| 危险色 | #F56C6C | 删除、举报、失物标签 |
| 信息色 | #909399 | 辅助文字、次要信息 |
| 背景色 | #F5F7FA | 页面底色 |
| 卡片白 | #FFFFFF | 卡片、表单背景 |
| 边框色 | #E4E7ED | 卡片边框、分割线 |

### 字体层级

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| H1 | 28px | 700 | 页面主标题 |
| H2 | 22px | 600 | 区块标题 |
| H3 | 18px | 600 | 卡片标题 |
| 正文 | 14px | 400 | 描述文字 |
| 辅助 | 12px | 400 | 时间、标签、次要信息 |

### 组件规范

| 组件 | 规范 |
|------|------|
| 卡片圆角 | 8px（小卡）/ 12px（大卡）|
| 阴影 | 0 2px 12px rgba(0,0,0,0.08) hover 态 |
| 按钮圆角 | 6px（小）/ 8px（标准）|
| 标签圆角 | 4px（直角标签）/ 20px（胶囊标签）|
| 输入框 | 边框 1px #DCDFE6，focus #409EFF + 光晕 |
| 头像 | 圆形，40/48/64px |

### 布局规范

| 规范 | 值 |
|------|------|
| 最大内容宽度 | 1200px |
| 页面内边距 | 24px |
| 卡片间距 | 16px |
| 顶部导航高度 | 60px |
| 网格列数 | 2 列（列表）/ 3 列（首页卡片）|

## 后续迁移建议

1. 每个 HTML 的 `<style>` 块可直接迁移到 Vue SFC 的 `<style scoped>`
2. Element Plus 组件标签替换为 `<el-xxx>` Vue 组件
3. 静态数据替换为 API 调用
4. `<a href>` 跳转替换为 `<RouterLink :to>`
5. 内联 Vue 实例替换为 Pinia Store
